# simple-retrieval
## A simple retrieval - java application example

## Environment

This example uses:
* Oracle JDK 11

## Build

* Gradle Install
https://gradle.org/releases/

* Build example this:
```
$ gradle build
$ java -jar build\libs\retrieval-example-1.0.jar
```
